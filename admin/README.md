# 코코아 관리자

- 프레임워크: Next.js
- 스타일링: Tailwind CSS
- 상태 관리: Zustand
- 라우팅: Next.js 라우터
- 데이터 패칭: React Query
- 폼 관리: React Hook Form
- 인증: Auth.js
- 데이터베이스: PostgreSQL
- 캐싱: Redis
- 언어: TypeScript 
- 빌드: Turborepo
- UI: Shadcn UI
- 분석 도구: Vercel Analytics
- 범용 훅: React Haiku

## 명령어

```bash
pnpm install
pnpm build
pnpm start
```
